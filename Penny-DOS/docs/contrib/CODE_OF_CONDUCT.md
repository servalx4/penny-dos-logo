# Code Of Conduct (for contributors, developers, and maintainers).

Thank you for contributing to Penny-DOS, we appreciate your efforts and contributions to the project. We expect all contributors to follow this code of conduct.

## Code of Conduct

1. Be respectful and considerate of others.
2. Avoid offensive language and behavior.
3. Respect intellectual property rights.
4. Do not engage in harassment or discrimination.
5. Do not engage in any form of bullying or harassment.
6. We keep the project free from any form of harassment or discrimination.
7. We have no secrets for developing. We are open source and transparent. We welcome feedback and suggestions from the community.
8. We are committed to maintaining a positive and inclusive environment for all contributors.
9. We encourage open communication and collaboration among contributors.
10. We value diversity and inclusivity in our community.
11. We strive to create a welcoming and supportive environment for all contributors.
12. We encourage constructive criticism and feedback to improve the project.
13. We value transparency and accountability in our community.
14. We encourage constructive criticism and feedback to improve the project.
